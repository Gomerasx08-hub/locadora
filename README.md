# locadora
 página de locadora de veículos



Passo 1: Meu primeiro passo foi adicionar um required em todos os campos no html, para se no caso de o usuario não inserir o o dado pedido nos campos dar erro.

Passo 2: Criei as tabelas necessárias para o banco de dados do site.

passo 3: Fiz a contagem de quantas paginas eu vou precisar de acordo com as informações dadas, e criei os files.

passo 4: 
Para o user story 1, fiz um metodo post com a rota chamada "/reserva". 
Const data vai guardar todos os emails, nomes, e as categorias. O insert into vai inserir essa informação no banco de dado. Se der certo, vai retornar o id da reserva, se não, vai retornar mensagem de erro.
